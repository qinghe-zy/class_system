# 接口文档说明

## 1. 统一返回结构

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

## 2. 认证机制

- 登录成功后返回 JWT
- 请求头携带：`Authorization: Bearer <token>`
- 通过拦截器完成接口权限校验与角色校验

## 3. 核心接口分组

### 认证
- `POST /api/auth/login`
- `GET /api/auth/me`

### 管理员
- 用户管理：`/api/admin/users/**`
- 课程管理/审核：`/api/admin/courses/**`
- 课堂内容审核：`/api/admin/course-contents/**`
- 会话总览：`GET /api/admin/sessions`
- 行为总览：`GET /api/admin/behavior/records`
- 签到惩罚总览：`GET /api/admin/signin/penalties`
- 统计：`/api/admin/stats/**`

### 教师
- 我的课程：`/api/teacher/courses/**`
- 课堂会话：`/api/teacher/sessions/**`
- 签到管理：`/api/teacher/signin/**`
- 附件上传：`POST /api/teacher/files/upload`
- 测验管理：`/api/teacher/quizzes/**`
- 课堂内容发布：`/api/teacher/course-contents/**`
- 行为记录：`GET /api/teacher/behavior/records`
- 统计：`/api/teacher/stats/**`

### 学生
- 课程列表与选课：`/api/student/courses/**`
- 在线课堂会话：`GET /api/student/sessions`
- 学生签到：`/api/student/signin/**`
- 学生测验：`/api/student/quizzes/**`
- 课堂内容查看：`GET /api/student/course-contents`
- 行为上报：`POST /api/student/behavior/report`
- 我的行为记录：`GET /api/student/behavior/records`
- 我的统计：`/api/student/stats/**`
