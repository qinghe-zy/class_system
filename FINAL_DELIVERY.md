# 最终交付说明

## 1. 本次迭代结果

本项目已整理为可直接交付的完整工程，包含：

- 前端 Vue3 工程
- 后端 Spring Boot 工程
- MySQL 建表与初始化脚本
- 启动脚本（Windows / Linux / macOS）
- 项目说明与开发文档

## 2. 与总目标对照

### 2.1 用户信息管理
- 已覆盖管理员对用户的增删改查、分页查询、角色分配、状态控制、密码重置。

### 2.2 课程录入功能
- 已覆盖教师建课、课程编辑、课程审核、课程发布与学生查看。

### 2.3 课堂互动功能
- 已覆盖课堂会话、签到、课堂测验、学生提交与教师查看。

### 2.4 行为检测功能
- 已覆盖学生端行为数据上报、后端规则分析、行为记录落库、教师/管理员/学生分角色查看。

### 2.5 数据统计与分析功能
- 已覆盖管理员平台统计、教师课程统计、学生个人统计与图表展示。

### 2.6 用户权限管理功能
- 已覆盖基于角色的菜单、接口、数据权限控制。

## 3. 可直接演示的主流程

1. 管理员登录并管理用户
2. 教师登录创建课程并发起课堂会话
3. 学生登录选课、签到、参与测验
4. 学生进入在线课堂后持续产生行为数据
5. 教师查看行为检测记录与统计结果
6. 管理员查看平台总览与审核内容

## 4. 运行要求

- JDK 21
- Maven 3.9+
- Node.js 20+ 或 22+
- MySQL 8.0+

## 5. 快速启动

### 数据库初始化
- Windows：`init-db.bat root 123456 ocms`
- Linux / macOS：`./init-db.sh root 123456 ocms`

### 启动后端
- Windows：`start-backend.bat`
- Linux / macOS：`./start-backend.sh`

### 启动前端
- Windows：`start-frontend.bat`
- Linux / macOS：`./start-frontend.sh`
