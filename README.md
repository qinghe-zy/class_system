# 在线课堂管理系统（重构交付版）

本版本以《在线课堂管理系统_可直接移植开发手册_终稿》为最高约束，对原压缩包进行了面向业务流程的重构。

## 本次交付重点

- 三端导航重构：管理员、教师、学生分别切换为“业务动作驱动”导航
- 前端语言重构：不再把课程 ID、会话 ID、资源 ID 作为主要操作入口
- 核心闭环保留并重组：
  - 用户与角色管理
  - 课程创建与课程审核
  - 资源上传与资源审核
  - 课堂活动、签到、测验
  - 统一学习器
  - 学习过程分析与风险说明
  - 平台/教师/学生统计页
- 视觉风格升级为轻量教育 SaaS 风格，统一了卡片、工作台、抽屉与表单交互

## 技术栈

- 前端：Vue 3 + Vite + Pinia + Element Plus
- 后端：Spring Boot + MyBatis-Plus + JWT（保持原项目技术主线）
- 数据库：MySQL（沿用原项目 SQL）

## 运行方式

### 前端

```bash
cd frontend
npm install
npm run dev
```

可选环境变量（推荐新增 `.env.local`，可参考 `frontend/.env.example`）：

- `VITE_API_BASE_URL`：后端根地址。为空时走同源路径（`/api`、`/uploads`）。
- `VITE_PROXY_TARGET`：本地开发代理目标（默认 `http://127.0.0.1:8080`）。
- `VITE_HTTP_TIMEOUT_MS`：通用接口超时。
- `VITE_UPLOAD_TIMEOUT_MS`：上传接口超时。

生产构建：

```bash
npm run build
```

### 后端

请在本地安装 Maven 后运行：

```bash
cd backend
mvn spring-boot:run
```

可选环境变量：

- `APP_UPLOAD_DIR`：自定义上传目录（默认项目根目录下 `uploads`）。

默认数据库配置见：

- `backend/src/main/resources/application.yml`
- `sql/schema.sql`
- `sql/data.sql`
